**替换transformers的源码内容，使其支持TBot-SA框架**

```1c
cd /vla/MyTBotSA1
conda activate tbot_sa1

# 1. 确认版本（必须是 4.57.1）
python -c "import transformers; print(transformers.__version__)"
# 2. 自动定位 transformers 安装目录（比硬编码 python3.10 更稳）
TRANSFORMERS_DIR="$(python -c "import transformers, os; print(os.path.dirname(transformers.__file__))")"
# 3. 备份原文件（推荐，方便回滚）
cp "${TRANSFORMERS_DIR}/models/qwen3_vl/modeling_qwen3_vl.py" \
   "${TRANSFORMERS_DIR}/models/qwen3_vl/modeling_qwen3_vl.py.bak"
# 4. 覆盖 patch（只替换 qwen3_vl 这一个文件）
cp src/lerobot/policies/TBot_SA1/transformers_replace/models/qwen3_vl/modeling_qwen3_vl.py \
   "${TRANSFORMERS_DIR}/models/qwen3_vl/modeling_qwen3_vl.py"
```


在 RoboTwin 2.0 和 LIBERO 中进行评估还需要它们的官方代码库。这些依赖项作为 Git 子模块包含在 `<path>` 下`third_party/`。要初始化它们，请运行：

```shell
# cd 各自空目录
cd /vla/MyTBotSA1/third_party/Depth-Anything-3
git clone https://github.com/ByteDance-Seed/Depth-Anything-3.git /vla/MyTBotSA1/third_party/Depth-Anything-3

git clone https://github.com/Lifelong-Robot-Learning/LIBERO.git /vla/MyTBotSA1/third_party/LIBERO
cd /vla/MyTBotSA1/third_party/RoboTwin
git clone https://github.com/RoboTwin-Platform/RoboTwin.git /vla/MyTBotSA1/third_party/RoboTwin
```
